# Olá, eu sou o Ricardo! 👋

### 🚀 Backend Developer | Java & Node.js Specialist | Mobile Enthusiast

Com mais de seis anos de experiência como desenvolvedor backend, especializei-me na criação de arquiteturas escaláveis e eficientes. Minha base sólida em **Java** e **Node.js/Express** me permite projetar sistemas capazes de gerenciar grandes volumes de tráfego e dados com alta performance.
