# 💫 About Me:
# Olá, eu sou o Ricardo! 👋

### 🚀 Backend Developer | Java & Node.js Specialist | Mobile Enthusiast

Com mais de seis anos de experiência como desenvolvedor backend, especializei-me na criação de arquiteturas escaláveis e eficientes. Minha base sólida em **Java** e **Node.js/Express** me permite projetar sistemas capazes de gerenciar grandes volumes de tráfego e dados com alta performance.

Além do backend, transito com fluidez no desenvolvimento mobile utilizando **Kotlin**, o que me dá uma visão 360º do ciclo de vida de um produto. Domino desde o design de **APIs RESTful** e gerenciamento de bancos de dados até a implementação em nuvem (**AWS/Azure**). 

Acredito que a tecnologia deve ser sustentável e comunicável; por isso, prezo por um código bem estruturado e pela capacidade de traduzir desafios técnicos complexos em soluções claras para todos os envolvidos no projeto.
